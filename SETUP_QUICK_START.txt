### Step 2: Get SMTP Credentials
Choose one (Mailtrap recommended):
- **Mailtrap** (https://mailtrap.io) - Sign up, create inbox, copy credentials
- **Gmail** - Enable 2FA, generate app password
- **SendGrid** (https://sendgrid.com) - Sign up, create API key
- **AWS SES** - Set up region, get SMTP credentials

### Step 3: Update Configuration
Edit `config/mail.php`:
```php
'smtp' => [
    'host' => 'smtp.mailtrap.io',  // Your provider's host
    'port' => 587,
    'username' => 'your_username',  // From provider
    'password' => 'your_password',  // From provider
    'encryption' => 'tls',
],
```
