## Common Questions

**Q: Can I link someone else's meter?**
A: Only if the account holder name matches or you have authorization

**Q: What if I enter the wrong account number?**
A: You'll need to delete and re-link with the correct number

**Q: How do I update my bill information?**
A: Bill data updates automatically from the utility provider (future feature)

**Q: Can I link meters from different cities?**
A: Yes! Link any meter you have access to

**Q: Why deactivate instead of delete?**
A: Deactivating keeps your history while hiding the meter from active view
